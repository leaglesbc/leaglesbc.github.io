C.R.A.P DESIGN | PROXIMITY

Proximity helps create organisation. By grouping similar elements together or in close proximity, you create a relationship between those elements. It also provides a focal point and can give the reader and idea of where they should start and finish reading.

HEADINGS AND IMAGES CREATE PROXIMTY
THROUGH BEING BIG AND NOTICABLE

PARAGRAPHS and SENTENCES are small. You can connect sentences through using Contrast and Proximity to make them stand out from headings. More sentences within proximity of each other, creates the feeling of importance and adds a focal point.

Proximity doesn't mean that elements have to be placed together, it means they should be visually connected in someway. This can be by use of point size, font, colour etc... Using CSS to have an image placed amongst the text with a text wrap or image float left ) uses Proximity to create a design feature.
